# Project-Slime-Heroes
A 2D platformer game developed with the [2D Engine](https://github.com/xpsa0421/2D-Game-Engine).

<br />

### Features
* 1 Main Menu, 1 Intro Scene, 2 In-Game Maps 
* 2D follow camera
* 2D collision handling
* UI and buttons
* Parallax background
* Sprite animation
* Sound
* Player input
